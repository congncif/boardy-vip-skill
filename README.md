# Boardy+VIP Agent Skill

A standardized AI Agent Skill for building modular iOS applications using the **Boardy** coordination framework and **VIP** (View-Interactor-Presenter) architecture.

## Overview

This skill enables AI coding assistants to:
1.  **Generate** clean, modular feature units (Microboards).
2.  **Enforce** architectural boundaries (Clean Architecture / DDD).
3.  **Optimize** build times using the Interface Module (IO) pattern.
4.  **Connect** modules seamlessly using the Global Plugin system.

## How to use

To give your AI agent these capabilities, simply copy the `boardy_vip` folder into your project's `.agents/skills/` directory.

```bash
cp -r boardy_vip /path/to/your/project/.agents/skills/
```

Once installed, your agent will automatically index the `SKILL.md` file and begin following these architectural standards for all feature development tasks.
